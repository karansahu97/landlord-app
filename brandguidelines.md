# Brand Guidelines

## Brand Personality
- Clean
- Trustworthy
- Structured
- Utility-focused
- Modern
- Calm
- Professional
- Designed for clarity & readability

---

## Color System

Primary Brand Color:
- Blue

Color Intent:
- Conveys trust, reliability, and financial clarity

Palette Structure:
- Primary Blue
- Blue Shades (Light / Dark Variants)
- Neutral Colors (White / Grayscale / Backgrounds)
- Functional Colors:
  - Success
  - Warning
  - Error

---

## Typography

Typeface:
- Inter

Usage Rules:
- Use Inter consistently across the product

Type Hierarchy:
- Headings (H1 – H6)
- Body Text
- Labels / UI Text
- Caption / Helper Text

Design Goal:
- High readability
- Clean visual rhythm
- Consistent spacing & alignment

---

## Spacing & Layout System

Base Unit:
- 4px

Rules:
- All padding, margins, gaps, and layout spacing must follow a 4px grid system
- Maintain consistent visual rhythm
- Ensure balanced layouts
- Optimized for mobile interfaces

---

## UI Style & Components

Visual Style:
- Minimal
- Modern
- Clean surfaces
- Soft contrast
- Subtle depth (if needed)

Component Consistency:
- Buttons
- Input Fields
- Cards
- Lists
- Navigation Elements

---

## Design Principles

- Prioritize simplicity & clarity
- Avoid visual clutter
- Focus on usability & legibility
- Maintain consistency across screens
